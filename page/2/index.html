Posted on 03-16-2018 In Tutorial

What is Hexo？

Hexo 是一个静态网站生成软件，它通过预定义的网页模板，把 Markdown 文件渲染为一个静态网页。除此之外，Hexo 还支持高度的自定义设置，通过应用社区共享的主题样式，你能轻易地实现网页的个性化。同时，我们可以把 Hexo 生成的静态网页部署到静态网站托管服务器上，例如：GitPage，让我们能快速搭建一个个人博客。
