网易云音乐无法联网

Posted on 03-20-2018 In Fixed

症状

前几天网易云突然不太能联网了，具体表现为：

  • 联网断断续续，通常刚开机可用，刚打开软件时可以播放一首歌
  • 有时可加载歌词，评论，封面，但无法播放
  • 网页版网易云也不能进入
  • 同一个局域网内的用户没有出现同样的问题
  • 分析流量，显示 connection refused 错误

尝试解决步骤

  • 清楚代理缓存
  • 检查 Hosts 是否被污染
  • IE 的高级功能里，恢复默认值
  • netsh winsock reset
  • 使用 win10 的网络重置功能
  • 检查网卡的 DNS 设置，如果有必要，手动指定 DNS
  • 卸载重装网易云

按照以上步骤，终于解决了问题，又可以随心所欲地听歌了
